# 뉴진스 팬레터 홈페이지 로그인 기능 추가

로그인된 유저가 팬레터 CRUD 작성하도록 구현

## 개발기간

23.11.29 ~ 23.12.04

## 주요기능

(1) 로그인/회원가입 화면 UI 작업  
(2) 로그인 화면 API 연결  
(3) Layout 컴포넌트 작성  
(4) 홈화면 UI 수정  
(5) json-server 셋업  
(6) 상세화면 UI 및 API 연결  
(7) 본인 게시물에만 수정,삭제 기능  
(8) 프로필 화면 UI  
(9) auth 서버에 변경된 닉네임 + 프로필이미지 저장

## 미구현

(1) accessToken 유효시간&유효여부 조정  
(2) json 서버에 변경된 닉네임 + 프로필이미지 저장
